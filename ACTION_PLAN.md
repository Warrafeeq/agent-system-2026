## 🔑 TOKEN SECURITY

### Creating Token Safely
```
✅ DO:
   - Create on YOUR GitHub account
   - Select only needed scopes
   - Don't share the token with others
   - Delete old tokens regularly

❌ DON'T:
   - Put token in code or commit
   - Share on public channels
   - Use in unsecured repositories
   - Leave token visible in terminal
```

### Token Scopes Explained
```
repo = Read/write code
workflow = Run GitHub Actions

Why both?
- repo = Access your code
- workflow = Run your workflows
```
