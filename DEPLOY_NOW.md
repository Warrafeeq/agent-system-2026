## 📋 EXACT STEPS

### ACTION 1️⃣: Create GitHub Token

**Open this URL:**
```
https://github.com/settings/tokens
```

**Then do this:**
1. Click "Generate new token"
2. Name it: `Agent-System-2026`
3. Check BOTH boxes:
   - ☑ repo
   - ☑ workflow
4. Scroll down and click "Generate token"
5. **COPY the entire token** (it looks like: `ghp_xxxx...`)
6. **Save it temporarily** (you'll use it next)

⚠️ **IMPORTANT**: You won't see the token again! If you close it, you'll need to create a new one.

### ACTION 2️⃣: Enable GitHub Actions

**Choose ONE option below:**

---

#### 🟢 OPTION A: Automated (Easiest - 1 minute)

**Open PowerShell and run:**
```powershell
cd C:\workspace
.\setup-actions.ps1
```

**Then:**
1. The script will ask you to paste the token
2. Right-click and select "Paste" (or Ctrl+V)
3. Press Enter
4. Script completes automatically ✅

**Done! Skip to ACTION 3**

---

#### 🔵 OPTION B: Manual Web UI (2 minutes)

**Step 1: Enable Actions**

Go to:
```
https://github.com/Warrafeeq/agent-system-2026/settings/actions
```

Do this:
1. Find the section: "Actions permissions"
2. Select the radio button: "⦿ Allow all actions and reusable workflows"
3. Scroll down and click "Save"
4. Wait for "Settings saved" message

**Step 2: Add Token Secret**

Go to:
```
https://github.com/Warrafeeq/agent-system-2026/settings/secrets/actions
```

Do this:
1. Click the green "New repository secret" button
2. Fill in:
   - **Name:** `GITHUB_TOKEN` (exactly this, uppercase)
   - **Value:** (paste your token from ACTION 1)
3. Click "Add secret"
4. Confirm it appears in the list
